## 实现说明

### Stage 1 (stage1_gradient.v)
- Sobel卷积计算水平和垂直梯度
- 梯度绝对值和归一化
- 邻域梯度最大值查找
- 窗口大小LUT查表

### Stage 2 (stage2_directional_avg.v)
- 窗口延迟对齐 (4 cycles)
- 核选择逻辑
- 5方向加权求和 (中心、上、下、左、右)
- 整数除法输出平均值

### Stage 3 (stage3_gradient_fusion.v)
- 2行梯度缓存 (FIFO结构)
- 5输入梯度排序网络
- 加权乘累加运算
- 除法输出融合结果

### Stage 4 (stage4_iir_blend.v)
- 混合比例选择
- IIR水平混合
- 窗口混合
- 最终输出混合
- IIR反馈控制

### 行缓存 (isp_csiir_line_buffer.v)
- 5行循环缓存
- 5x5窗口生成
- IIR反馈写回支持
- 边界处理
