## 关键决策
| 决策点 | 选择 | 原因 |
|--------|------|------|
| 定点格式 | 纯整数格式 (U10.0, U14.0, U20.0) | 算法特性为纯整数运算，简化硬件实现 |
| Stage 3 除法 | 移位近似 | 除数范围有限，精度损失可接受 |
| 误差容忍 | 平均误差 < 2 LSB | 满足图像处理视觉质量要求 |

## 数据依赖关系
| 阶段 | 依赖 | 说明 |
|------|------|------|
| Stage 1 | 5x5窗口 -> 梯度 -> 窗口大小 | 无跨行依赖 |
| Stage 2 | 5x5窗口 + win_size -> 方向平均 | 无跨行依赖 |
| Stage 3 | avg* + 梯度(上一行) -> 融合 | 需要梯度行缓存 |
| Stage 4 | blend* + avg_u(上一行) -> 输出 | **IIR反馈核心** |

## 行缓存需求
- 像素行缓存: 4行 x 10-bit
- 梯度行缓存: 1行 x 14-bit
- IIR行缓存: 2行 x 20-bit
- **总计: 约 498 Kbits**
