# isp-csiir 项目 Debug 记录

## 问题记录

### 2026-03-22 梯度归一化精度问题（严重）✓ 已修正
- 现象: 除5近似为除4，引入 25% 系统偏差
- 定位: Stage 1 梯度计算, rtl/stage1_gradient.v 第157行
- 原因: rtl-algo 未评估精度影响，rtl-arch 未审核
- 影响: 窗口大小 LUT 阈值等效降低，边缘检测过度敏感
- 方案: 改用乘法近似 (×205 >> 10)，误差降至 0.1%
- 状态: ✓ 已修正

### 2026-03-22 Stage 3 变量除法实现错误（严重）✓ 已修正
- 现象: RTL 直接用 `/` 操作符实现变量除法
- 定位: stage3_gradient_fusion.v 第 404-407 行
- 原因: 架构设计了迭代除法器，但 RTL 未实现
- 影响: 600MHz 时序无法收敛，组合逻辑除法器面积巨大
- 方案: 使用 256 entries 压缩 LUT 实现单拍除法
- 状态: ✓ 已修正

### 2026-03-22 LUT 索引重叠 Bug（严重）✓ 已修正
- 现象: LUT 索引压缩逻辑有重叠，导致巨大误差
- 定位: stage3_gradient_fusion.v 索引压缩逻辑
- 原因: grad_sum 1-1023 映射到 index 1-255，与高位范围重叠
- 影响: 误差高达 1000+ LSB，结果完全错误
- 方案: 重新设计无重叠索引压缩
- 状态: ✓ 已修正
